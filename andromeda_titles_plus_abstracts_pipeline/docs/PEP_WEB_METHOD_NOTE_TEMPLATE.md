# PEP-Web-style methods note template

Bibliometric metadata were analysed using a metadata-first, human-in-the-loop Andromeda workflow. The analysis used article titles and abstracts as the textual layer. No PDFs were mirrored and no full-text article corpus was redistributed.

Candidate terms were extracted from normalized titles and abstracts with n-gram TF-IDF. Terms were then mapped to semantic concepts using an auditable semantic map. Ambiguous terms were placed in a manual review queue. Trend analyses were calculated at article level, with each concept counted once per article.

The results should be interpreted as a discourse map of the title/abstract layer, not as full-text analysis.
