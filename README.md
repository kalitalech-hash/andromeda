# andromeda
Bibliometric research in psychotherapy

A computational toolkit for bibliometric and discourse analysis of psychotherapy literature.

## Author

Lech Kalita

## Computational research assistant

Andromeda Nowicka (experimental digital research agent)

## Project description

This project contains scripts used to analyze the evolution of clinical discourse in psychotherapy and psychiatry journals.

Methods include:

- automated web scraping
- keyword normalization
- semantic merging
- co-word analysis
- temporal trend analysis
- topic clustering

The repository accompanies the following studies:

- Bibliometric analysis of the journal *Psychoterapia*
- Bibliometric analysis of the journal *Psychiatria Polska*
