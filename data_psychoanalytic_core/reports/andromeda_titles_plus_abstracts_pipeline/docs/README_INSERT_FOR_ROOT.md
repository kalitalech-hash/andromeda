Pipeline for title + abstract discourse mapping. It extends title-based analysis to openly available or otherwise lawfully accessible abstracts, with metadata-first acquisition, no PDF mirroring, auditable candidate-term extraction, semantic normalization, periodization, trend analysis, and co-occurrence networks. It is suitable for PEP-Web-style psychoanalytic corpora when restricted source text is kept outside the public repository and only derived analytical outputs are published.
