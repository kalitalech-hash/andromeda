# Methodological note

This pilot uses a metadata-first, human-in-the-loop workflow to map the publication output of people publicly listed by PTPP.

The unit of membership evidence is a public PTPP page snapshot, not a direct survey or institutional confirmation. Membership categories must be preserved as separate analytical fields. A publication is attributed to a PTPP-listed person only after author disambiguation. The project does not infer that every publication represents an official PTPP position.

Recommended interpretation wording:

- "among publications attributed to persons listed in the public PTPP membership snapshot"
- "in the bibliographic metadata layer"
- "based on title/abstract/keyword metadata"
- "publication output linked to PTPP-listed persons"

Avoid:

- "PTPP published..."
- "PTPP caused..."
- "the Society's scientific output" unless institutional authorship is explicitly documented.
